# Examen modular 2: Cloud Infrastructure & AWS

## Parte A: Conceptual

1. Explique la diferencia entre región y zona de disponibilidad.
2. Diseñe una VPC con subred pública y privada para una aplicación web.
3. ¿Qué problema resuelve Auto Scaling?
4. Compare S3, EBS y RDS desde el punto de vista de almacenamiento.
5. Explique CloudWatch y CloudTrail con ejemplos de uso.
6. Diseñe una política de menor privilegio para una aplicación que escribe logs y lee secretos.

## Parte B: Diseño

Entregue un diagrama de arquitectura AWS para la aplicación del módulo 1, incluyendo red, cómputo, datos, seguridad, monitoreo y costos.
