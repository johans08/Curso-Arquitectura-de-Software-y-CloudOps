# Preguntas frecuentes

## ¿Debo usar GitHub obligatoriamente?

Sí. Las tareas deben estar en GitHub. Classroom se utiliza temporalmente para registrar el enlace de entrega.

## ¿Puedo entregar capturas en lugar de código?

No como evidencia principal. Las capturas pueden complementar, pero la evidencia central es el repositorio.

## ¿El frontend debe ser JavaScript?

No. Para mantener el curso simple y coherente con .NET, el frontend de referencia usa Blazor.

## ¿Se requiere una cuenta AWS personal?

Depende de la modalidad del grupo. Puede usarse AWS Academy, AWS Educate o una cuenta personal con presupuesto controlado.

## ¿Qué pasa si no puedo ejecutar recursos pagados?

El estudiante puede documentar el diseño, usar capturas de simulación, revisar plantillas IaC y trabajar con alternativas gratuitas o locales cuando el instructor lo autorice.
