# Glosario técnico

- **Arquitectura de software:** estructura de alto nivel de un sistema, sus componentes, relaciones y decisiones principales.
- **Clean Code:** conjunto de prácticas para escribir código legible, mantenible y expresivo.
- **SOLID:** cinco principios de diseño orientado a objetos para reducir acoplamiento y mejorar extensibilidad.
- **API:** interfaz que expone capacidades del sistema a otros consumidores.
- **DTO:** objeto usado para transferir datos entre capas o procesos.
- **JWT:** token firmado que transporta claims para autenticación/autorización.
- **VPC:** red virtual aislada dentro de AWS.
- **EC2:** servicio de cómputo basado en instancias virtuales.
- **S3:** almacenamiento de objetos en AWS.
- **RDS:** bases de datos administradas en AWS.
- **CloudWatch:** servicio de métricas, logs y alarmas.
- **CloudTrail:** registro de actividad de API y auditoría.
- **IaC:** infraestructura como código.
- **CI/CD:** integración continua y entrega/despliegue continuo.
- **Blue/Green:** estrategia de despliegue con dos ambientes equivalentes.
- **Canary:** estrategia de despliegue gradual hacia un pequeño porcentaje de usuarios.
