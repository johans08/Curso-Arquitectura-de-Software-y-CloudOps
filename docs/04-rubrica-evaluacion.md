# Rúbrica de evaluación del Módulo 1

## Niveles

| Nivel | Descripción |
|---|---|
| Excelente | Entiende el concepto, lo aplica correctamente y justifica decisiones |
| Bueno | Implementa correctamente, aunque con justificación parcial |
| Básico | Logra que funcione, pero con bajo diseño |
| Insuficiente | No evidencia comprensión o incumple restricciones técnicas |

---

## Evaluación semanal

| Criterio | Excelente | Bueno | Básico | Insuficiente |
|---|---|---|---|---|
| Concepto | Explica con claridad y ejemplos propios | Explica con ejemplos simples | Repite definiciones | No logra explicar |
| Diseño | Hay límites claros y bajo acoplamiento | Diseño aceptable | Diseño mezclado | Sin diseño |
| .NET | Código limpio y organizado | Código funcional | Código funcional pero desordenado | No compila |
| SQL Server | Modelo consistente con constraints | Modelo funcional | Modelo incompleto | Sin integridad |
| Documentación | Incluye diagramas y ADR | Incluye README | README superficial | Sin documentación |
