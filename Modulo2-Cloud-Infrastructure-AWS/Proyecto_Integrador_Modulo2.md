# Proyecto integrador del Módulo 2

## Nombre

Diseño y despliegue de arquitectura web en AWS

## Objetivo

Diseñar una arquitectura AWS para desplegar la aplicación .NET del módulo 1 con buenas prácticas de red, cómputo, almacenamiento, base de datos, monitoreo, seguridad y costos.

## Componentes mínimos

- Diagrama de VPC con subredes públicas y privadas.
- Diseño de cómputo con EC2/Auto Scaling o alternativa aprobada.
- Almacenamiento S3 para objetos.
- Base de datos administrada RDS o diseño equivalente documentado.
- CloudWatch para métricas/logs/alertas.
- CloudTrail para auditoría.
- IAM roles y políticas mínimas.
- Estimación básica de costos y estrategia de apagado para prácticas.

## Entregables

- Documento de arquitectura.
- Diagrama Mermaid, draw.io o PNG.
- Checklist de seguridad.
- Checklist de costos.
- Evidencia de recursos creados o simulación aprobada.
- Plan de despliegue y rollback.

## Rúbrica

| Criterio | Ponderación |
|---|---:|
| Diseño de red y seguridad | 25% |
| Cómputo, almacenamiento y base de datos | 25% |
| Observabilidad y auditoría | 20% |
| Costos y operación | 15% |
| Documentación y defensa técnica | 15% |
