# Módulo 2: Cloud Infrastructure & AWS

## Objetivo del módulo

Dominar el despliegue de infraestructura profesional en AWS, conectando decisiones de arquitectura con red, cómputo, datos, observabilidad, seguridad y costos.

## Producto del módulo

Despliegue controlado de la aplicación .NET en una arquitectura AWS con red, cómputo, base de datos, monitoreo y seguridad básica.

## Competencias esperadas

Al finalizar este módulo, el estudiante debe poder:

- Explicar decisiones técnicas con lenguaje profesional.
- Diseñar soluciones con diagramas y documentación clara.
- Implementar prácticas alineadas con operación real.
- Justificar trade-offs de costo, seguridad, mantenibilidad y escalabilidad.
- Subir evidencias completas a GitHub.

## Evaluación del módulo

- Prácticas semanales.
- Examen modular.
